Netcat -

Netcat（通常缩写为nc）是一种用于使用TCP或UDP读取和写入网络连接的计算机网络实用程序。Netcat被设计为可靠的后端，可以直接或轻松地由其他程序和脚本驱动。同时，它是一个功能丰富的网络调试和调查工具，因为它可以产生几乎任何一种用户可能需要的连接，并具有多种内置功能。

其功能列表包括端口扫描，传输文件和端口监听，可用作后门。

命令提示符中同时运行多命令 link

先后运行多个不同程序
如果在命令行提示符窗口中需要执行多个命令，那么可以同时输入各个命令，让它们一个接着一个地运行。此时我们需要用到“&&”和“||”来组合两条命令，其含义为：
        a&&b 先执行a命令，成功后再执行b命令。
        a||b 先执行a命令，若执行不成功则执行b命令，若执行成功则不再执行b命令。
例如，在命令行提示符后输入cd..&&dir，回车后首先执行返回上级目录的命令，然后再进行文件和文件夹列表显示。
